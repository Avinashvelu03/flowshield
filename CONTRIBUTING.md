## Getting Started

1. Fork & clone this repo
2. `npm install` or `bun install`
3. `git checkout -b feat/your-feature`

## Development

- Tests: `npm test` or `bun test`
- Build: `npm run build`

## Guidelines

- TypeScript strict mode
- Zero external dependencies
- Composable, edge-runtime compatible patterns
- Add tests for all new functionality

## Reporting Issues

Use [issue templates](https://github.com/Avinashvelu03/flowshield/issues/new/choose).
